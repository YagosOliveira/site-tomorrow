# Tomorrowland Inspired Website

Este projeto é um site inspirado no festival de música Tomorrowland, criado com o intuito de aplicar conhecimentos teóricos e práticos de HTML, CSS e JavaScript. O site visa capturar a essência vibrante e dinâmica do festival, enquanto serve como uma vitrine para técnicas modernas de desenvolvimento web.

## Sobre o Projeto

O objetivo deste site é criar uma experiência imersiva que lembra os visitantes da magia e energia do festival Tomorrowland. Usando tecnologias web modernas, o site apresenta informações sobre o festival, artistas, datas de eventos e galerias de fotos, tudo isso através de uma interface visualmente atraente e interativa.

### Funcionalidades

- **Home Page:** Uma introdução visualmente rica ao mundo do Tomorrowland.
- **Linha do Tempo do Evento:** Uma timeline interativa dos eventos e principais atrações do festival.
- **Galeria de Fotos:** Uma coleção de imagens vibrantes do festival.
- **Informações do Artista:** Perfis e informações sobre os artistas participantes.
- **Contato e Informações:** Formulário de contato e informações sobre ingressos e acesso.

### Tecnologias Utilizadas

- **HTML:** Estrutura básica do site.
- **CSS:** Estilização customizada, incluindo layouts flexíveis e responsivos.
- **JavaScript:** Interatividade do site, manipulações do DOM e animações.
